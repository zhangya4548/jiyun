
## 集运系统,集运下单系统,国际物流集运系统,国际仓储系统,转运系统源码出售

详细功能说明   

**包裹管理**

会员可以预报包裹，入库后可以对包裹申请各种增值服务如分箱，合箱，拍照，清点、选择发货渠道，创建运单等等。
管理员可以用扫描枪扫描入库，同步打印出包裹标签，若客户未预报包裹，可以自动代为客户预报包裹，然后扫描包裹标签号与仓库货架号上架包裹。
**运单管理**

会员可以自行创建已入库的预报包裹
会员可以直接下单，针对门店客户
会员可以通过EXCEL批量导入运单
管理员也可以在后台导入，创建运单
各种与运单有关的功能，如扣费、打印运单、批次管理、发货（运单国际段状态更新）、运单导入导出、国内快递导入、取件管理等等

**内容管理**

文章管理系统、在线留言、友情链接、导航管理、各种前台内容设置管理（包含幻灯片、联系方式等）

**会员管理**

会员基本信息管理（会员姓名、入库识别码、手机、邮件、地址等联系方式管理）
会员等级管理，根据会员等级设置运费折扣
会员地址簿管理（分收件人管理，发件人管理；收件人每个地址可选择性的绑定身份证，并且还可以批量上传地址薄导入，一次性可添加无数个地址，快捷方便。）
各种与会员有关的功能，如扣费、充值、查看各种记录、发通知信息、修改删除等等

**财务管理**

财务流水、货币汇率设置、支付方式（在线充值、银行转账）、会员积分、理赔管理

**转运设置**

运单状态设置、仓库设置、发货渠道设置、国内快递设置、运单附加服务设置、发货状态设置、报关条码、国内快递单号等

**系统设置**

系统常用的基本设置、管理员后台管理权限设置以及邮件/短信发送通知设置等

  核心功能模块   

**一单到底服务模式**
后台管理员在对应渠道开启“一单到底”服务模式，并导入相应的报关条码和国内快递单号，客户在前台和管理员在后台创建运单的同时，系统都可以匹配报关条码和国内快递单号。

**扫描枪**
入库和出库设置扫描枪自动识别，加快操作速度，减少操作失误
扫描枪节点（入库、出库）

**批量扣款**
支持批量扣款

**全球仓储系统**
支持不同国家间仓储存储，后台可以设置仓库仓位，方便操作员包裹查询；支持不同货币换算、重量换算，真正实现一站多国家使用。

**在线报价、运单跟踪**
全自动化，减少客服人数为您节省更多开支；运单除了显示系统物流状态外还可自动显示快递公司物流状态。

**针对大宗客户**
支持批量快递导入运单功能，大客户只需把所有国内收货地址整理在Excel中然后发给仓库审核并导入。

**批量入货或扫描入库**
能看到客户提交的运单信息，待包裹入库当日，即刻按照运单要求操作并能批量入货或扫描入库。

**多种批量导入**
多种批量导入，导出功能，操作简单方便，提高效率。

**地址薄**
分收件人管理，发件人管理；收件人每个地址可选择性的绑定身份证，并且还可以批量上传地址薄导入，一次性可添加无数个地址，快捷方便。

**在线支付**
支持多个在线支付平台（支付宝、微信、paypal、网银支付接口）或者联系员工代理充值。

**多种计费方式
比如A类、B类、C类；每个发货方式设置不同的资费标准。比如首重费用、续重、税费率，三级计费模式。**（计费公式参考：计费功能升级说明）。

**不同的收费标准**
能针对不同客户等级单独设置不同的收费标准，比如某VIP客户，普通客户。

**不同的计重模式**
可以设置进位取整、0.5进制、 实际计重、退位取整4种计重模式。

**员工分配范围权限**
管理员能给每个员工分配范围权限；员工只能看到自己工作范围内的功能。

**仓库权限分配**
支持多个仓库并且可以设置员工对不同仓库的管理权限

**理赔申请**
客户可以因为某些情况在线申请理赔，简洁方便。

**上门取件申请**
在些情况需要上门取件时，可以让仓库员人过去取件。

**物流跟踪信息查询**
支持国内，国外所有快递物流信息跟踪查询，随时让客户知道运单目前位置及状态。

**报关条码管理系统**
批量导入报关条码到系统保存，需要使用时后台设置自动匹配运单

**文章管理系统**
可以发布费用时效、行业新闻、帮助中心

**数据统计**
入库/出库统计 | 营利统计 | 会员转运统计 | 信息数统计 | 会员数统计 | 充值记录统计

**留言管理**
用户有什么问题都可在线留言反馈到后台。
